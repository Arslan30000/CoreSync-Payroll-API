# CoreSync Employee Payroll & Leave Management System - API

This repository contains the RESTful PHP backend and MySQL database schemas powering the CoreSync Employee Payroll & Leave Management System[cite: 1]. It serves JSON data to the Java Swing client application.

##  Tech Stack
* **Server:** Apache (via XAMPP)
* **Database:** MySQL
* **Scripting:** PHP (PDO for secure database interactions)

##  Local Development Setup (XAMPP)
1. Install and launch **XAMPP**. Start the **Apache** and **MySQL** modules.
2. Navigate to your XAMPP `htdocs` directory (usually `C:\xampp\htdocs`).
3. Clone this repository directly into `htdocs`:
